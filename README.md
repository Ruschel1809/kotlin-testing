# Kotlin Testing

Übungen und Case Study zu Unit Testing in Kotlin aus der Weiterbildung zur
**KI-Entwicklung** (Juni – Dezember 2025).

## Inhalte

| Ordner | Beschreibung |
|---|---|
| `KotlinUebung5101Testing` | Wettervorhersage – JUnit5 Tests mit `assertEquals` und `assertThrows` |
| `KotlinUebung5102` | Temperaturkonverter – Tests mit Floating-Point-Delta und Grenzwerten |
| `CaseStudy5201Testing` | Task-Manager – CRUD Tests mit JUnit5 und MockK Framework |

## Highlights
- Floating-Point-Tests mit Delta-Toleranz
- Exception-Testing mit `assertThrows`
- Mock-Objekte mit **MockK** Framework
- `@BeforeEach` für saubere Testisolation
- Vollständige CRUD-Abdeckung inkl. Fehlerfälle

## Technologien
- **Sprache:** Kotlin
- **Frameworks:** JUnit5, MockK
- **Build:** Gradle
- **IDE:** IntelliJ IDEA
